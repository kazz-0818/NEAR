# Google OAuth（ユーザー連携）— カレンダー含む

- LINE で **「Google連携」** と送ると、ブラウザ用の許可 URL が返る。
- スコープには **スプレッドシート／Drive 検索**に加え、**Google カレンダー（primary の予定一覧・追加）**用の `calendar.events` が含まれる。
- **既に連携済み**でカレンダーだけ使えない場合は、同意時にカレンダー権限が無かった可能性がある。**もう一度「Google連携」**して再許可する。
- GCP では **Google Calendar API** を有効にし、OAuth 同意画面のスコープにカレンダー系が出ることを確認する。
